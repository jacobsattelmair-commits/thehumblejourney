Apple Health

Apple Health does not have a web API, so use the free Health Auto Export app to send data to this dashboard automatically every day.

  1. Download Health Auto Export from the App Store (free)
  2. Open the app and go to Automations then Add Automation
  3. Choose REST API as the export type
  4. Set the URL to:
  5. Add a header: Key = Authorization, Value = Bearer YOUR_SECRET (get this secret from your Netlify dashboard under Environment variables, APPLE_HEALTH_SECRET)
  6. Select metrics: Steps, Active Energy Burned, Resting Heart Rate, VO2 Max
  7. Set schedule to Daily and tap Save
